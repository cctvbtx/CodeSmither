数据字典 SimpleCrm

表 : admin = 管理员表

字段名 数据类型 列长 默认值 非空 自动递增 备注
id INT 10 是 自增 主键自增ID
username VARCHAR 32 是 用户名
password VARCHAR 32 是 密码（密文）
nickname VARCHAR 32 昵称
is_super BIT 1 是否是超级管理员
status BIT 0 状态
ontid VARCHAR 64 区块链识别ID
ont_image VARCHAR 255 ontid二维码图片地址
delete_flag BIT 0 删除标记
create_time DATETIME 19 是 创建时间
update_time DATETIME 19 是 更新时间

表 : land = 地块信息表

字段名 数据类型 列长 默认值 非空 自动递增 备注
id INT 10 是 自增 主键自增ID
ontid VARCHAR 64 区块链识别ID
ont_image VARCHAR 255 ontid二维码图片地址
name VARCHAR 32 名称
area FLOAT 12 面积（平方米）
number VARCHAR 64 是 地块编号
address VARCHAR 64 地址信息
other TEXT 65,535 other
delete_flag BIT 0 删除标记
create_time DATETIME 19 是 创建时间
update_time DATETIME 19 是 更新时间

表 : land_operate = 地块操作记录表

字段名 数据类型 列长 默认值 非空 自动递增 备注
id INT 10 是 自增 主键自增ID
land_id INT 10 是 地块id
user_id INT 10 是 操作人员ID
operate VARCHAR 32 是 操作事项
delete_flag BIT 0 删除标记
ont_hash VARCHAR 255 存证hash
remark VARCHAR 255 备注
other TEXT 65,535 其他
create_time DATETIME 19 是 创建时间
update_time DATETIME 19 是 更新时间

表 : payroll = 工资记录表

字段名 数据类型 列长 默认值 非空 自动递增 备注
id INT 10 是 自增 主键自增ID
user_id INT 10 是 操作人员ID
ont_hash VARCHAR 255 存证hash
other TEXT 65,535 其他
work_time FLOAT 12 是 工作时长(小时)

update_time DATETIME 19 是 更新时间
